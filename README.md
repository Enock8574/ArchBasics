# ArchBasics
Script para Archlinux o derivadas que contienen paquetes basicos para ser instalados

Paquetes incluidos
1.htop
2.neofetch
3.nano
4.wget
5.gufw
6.gparted
7.curl
8.pipewire
9.wireless_tools
10.wireless-regdb

A su vez el script actualiza todos los paquetes que viene por defecto en el sistema
